If you install Tomcat 7 using apt:sudo apt-get install tomcat7
Then the webapps are located in /var/lib/tomcat7

N.B. dare al file axis2.war i permessi chmod 777 per installare axis dopo aver copiato il war nella cartella webapps

il webservice .class deve essere messo in una cartella con lo stesso nome della classe così:

es: ciaoMondo -> ciaoMondo.class, META-INF->services.xml

cambiare il proprietario ed il gruppo della cartella in tomcat8: sudo chown -R tomcat8:tomcat8 ciaoMondo/

N.B.

Per caricare un file html o php andare su /var/www/html

dare i permessi con chmod 777 nomefile.html

per rebootare apache: sudo /etc/init.d/apache2 restart (per tomcat: /etc/init.d/tomcat8 restart )
